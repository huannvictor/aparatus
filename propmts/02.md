# Requisitos Funcionais
- Deve ser exibido o botão de login se o usuário estiver deslogado. Se estiver logado use o avatar, nome e email (hard-coded).
- Botão de 'agendamentos' deve levar para '/bookings'.
- Botões das categorias (cabelo, barba etc.) devem levar para '/barbershops?search={termo}'. Veja um exemplo em `/components/quick-search.tsx`.
- Use o componente Sheet do shadcn/ui para exibir o menu.
- Botão 'sair da conta' não deve fazer nada, por enquanto. Code apenas a interface.
- Este componente deve ser aberto ao clicar no botão 'menu' do `/components/header.tsx`.
- o componente "sair da conta" ainda não está feito, faça.
- confira de todas as cores do componente `/components/menu.tsx` estão com as cores, espaçamentos e posicionamento corretos.
